# FreelanceARM
(Сюзев Михаил ИВТ-25-1Б, Исмаилов Рамал ИВТ-25-1Б)

FreelanceARM представляет собой автоматизированное рабочее место для работы с биржей фриланса.

Приложение моделирует основные процессы фриланс-платформы: регистрацию пользователей, работу в роли покупателя и фрилансера, заполнение анкеты, поиск исполнителей, создание заказов, чат, прикрепление файлов, завершение заказов и выставление рейтинга.

---

## Назначение проекта

Цель проекта — упростить взаимодействие между покупателем и фрилансером.

Покупатель может быстро найти подходящего исполнителя по навыкам, бюджету и времени работы. Фрилансер может создать анкету, указать свои навыки, ставку, рабочее время и описание.

Проект разработан как учебная модель фриланс-биржи и демонстрирует работу с графическим интерфейсом, локальным хранением данных, пользовательскими ролями, заказами и сообщениями.

---

## Основные возможности

### Аккаунты пользователей

В программе реализована регистрация и авторизация пользователей.

При регистрации пользователь указывает:

- имя пользователя;
- логин;
- пароль;
- фотографию профиля.

Один и тот же аккаунт может использоваться в двух ролях:

- покупатель;
- фрилансер.

Роль выбирается при входе в приложение.

---

### Возможности фрилансера

Фрилансер может:

- заполнить анкету;
- указать навыки;
- указать ставку;
- выбрать время работы;
- отметить режим «Круглосуточно»;
- добавить описание о себе;
- сохранить или удалить анкету;
- просматривать список покупателей;
- вести чат с покупателями;
- получать заказы;
- получать оценки после завершения заказов.

---

### Возможности покупателя

Покупатель может:

- указать нужный навык;
- указать бюджет;
- выбрать нужное время выполнения работы;
- включить режим «Круглосуточно»;
- найти подходящих фрилансеров;
- выбрать исполнителя из списка;
- начать заказ;
- вести чат с фрилансером;
- прикреплять файлы к сообщениям;
- завершить заказ;
- поставить оценку фрилансеру.

---

### Поиск фрилансеров

Поиск исполнителей учитывает:

- совпадение навыков;
- бюджет покупателя;
- рабочее время фрилансера;
- режим «Круглосуточно»;
- рейтинг фрилансера.

Если у фрилансера указано «Круглосуточно», он подходит под любое нужное время.

Собственная анкета пользователя не отображается в списке фрилансеров, поэтому пользователь не может заказать работу сам у себя.

---

### Заказы

Покупатель может начать заказ у выбранного фрилансера.

После начала заказа:

- создается новый заказ;
- открывается чат;
- в чате появляется системное сообщение «Заказ открыт».

После завершения заказа:

- заказ получает закрытый статус;
- покупатель ставит оценку;
- рейтинг фрилансера пересчитывается;
- обновляется статистика;
- в чате появляется системное сообщение «Заказ закрыт».

---

### Чат

В программе реализован чат между покупателем и фрилансером.

В чате можно:

- отправлять текстовые сообщения;
- прикреплять файлы;
- открывать прикрепленные файлы;
- видеть системные сообщения о статусе заказа.

Сообщения сохраняются и не пропадают после перезапуска приложения.

---

### Профиль пользователя

В окне «Мой профиль» отображаются:

- фотография;
- имя пользователя;
- логин;
- рейтинг;
- количество выполненных заказов;
- количество покупок.

Также в этом окне доступны:

- выход из аккаунта;
- удаление аккаунта с подтверждением паролем.
