# Биржа фриланса

Учебный Qt Widgets/qmake проект: одно приложение с двумя ролями — покупатель и фрилансер.

## Запуск

1. Открыть `FreelanceARM.pro` в Qt Creator.
2. Выбрать комплект `Desktop Qt 5.12.12 MinGW 64-bit`.
3. Нажать `Run qmake`.
4. Нажать `Clean` и `Rebuild`.
5. Запустить проект.

## Возможности

- Регистрация и авторизация пользователей.
- Выбор роли: покупатель или фрилансер.
- Уникальные логины для всех ролей.
- Выход из аккаунта и вход в другой аккаунт.
- Анкета фрилансера: имя, навыки, ставка, рейтинг, часы работы, описание, фото профиля.
- Интерфейс покупателя: создание задачи, список фрилансеров, карточка выбранного исполнителя, чат.
- Интерфейс фрилансера: анкета, список покупателей, чат с выбранным покупателем.
- Чат между конкретным покупателем и конкретным фрилансером.
- Данные сохраняются локально в JSON-файлах в папке `FreelanceARMData`.

## Файлы данных

Папка данных создается автоматически в стандартной папке приложения пользователя.
В ней находятся:

- `users.json`
- `freelancers.json`
- `tasks.json`
- `messages.json`

Пароли сохраняются не в открытом виде, а как SHA-256 хэш.
